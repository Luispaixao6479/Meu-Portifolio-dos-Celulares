Imagem do Projeto 1: O Celular da Iphone

O projeto é uma página criada pelo HTML e CSS, destinada a exibir celulares. O objetivo é criar os logotipos dos aplicativos, o telefone, o contato, a mensagem, a câmera e a galeria fornecendo uma descrição breve de cada um.

A estrutura do site é baseada em HTML, que define a semântica e o layout do contéudo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o contéudo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e de detalhes dos celulares em uma seção flexível (.celular). Cada celular é destacado em seu próprio container (.celular), mostrando pelas imagens dos celulares. O celular tem os aplicativos e os jogos. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos da tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada pelo Google Fonts e aplicando um esquema de cores suaves. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e de esquema de cores contribui para a atmosfera acolhedora do site, incentivando a criação de imagens dos celulares.
